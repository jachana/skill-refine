# refine

Backlog refinement / grooming / enrichment — make a backlog ready to work: enrich thin cards with context and acceptance criteria, estimate, order by priority, and flag what's blocked, duplicated, or…

## Install (Claude Code)

```
/plugin marketplace add jachana/claude-marketplace
/plugin install refine@jachana
```

Part of the [jachana marketplace](https://github.com/jachana/claude-marketplace).
